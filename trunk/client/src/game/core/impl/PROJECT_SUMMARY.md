/**
 * 核心逻辑文件
 * ─────────────────────────────────────────────────────────────────────────
 * 
 * 1. GameState.ts (200+ 行)
 *    - GameState 类：游戏状态管理
 *    - 功能：帧号记录、状态控制、时间追踪、数据存储
 *    - 接口：GameSnapshot（状态快照）
 * 
 * 2. Actor.ts (300+ 行)
 *    - Actor 基类：游戏世界中的所有对象
 *    - 继承关系：Unit、Building 等
 *    - 功能：位置、旋转、缩放、生命值、状态管理
 *    - 方法：update()、fixedUpdate()、移动、伤害处理
 * 
 * 3. GameSystem.ts (500+ 行)
 *    - GameSystem 基类：游戏系统框架
 *    - 内置系统：
 *      * MovementSystem - 单位移动
 *      * DamageSystem - 伤害计算
 *      * EventSystem - 事件分发
 *    - Game 主类：整合所有系统和角色
 *    - 特性：支持服务器/客户端模式
 * 
 * 4. Unit.ts (200+ 行)
 *    - Unit 类：继承 Actor 的具体单位实现
 *    - Building 类：建筑实现
 *    - 功能：技能、等级、经验系统
 * 
 * 5. SceneManager.ts (150+ 行)
 *    - SceneManager 类：场景管理器
 *    - 功能：根据关卡配置和地图配置初始化场景
 *    - 支持动态创建和管理角色
 * 
 * 6. GameRunner.ts (300+ 行)
 *    - ClientGameRunner：客户端游戏运行器
 *      * 整合 Game 和 World（渲染）
 *      * 处理渲染同步
 *    - ServerGameRunner：服务器游戏运行器
 *      * 纯逻辑处理
 *      * 帧同步和状态快照
 * 
 * 7. GameUtils.ts (400+ 行)
 *    - DistanceUtils：距离和方向计算
 *    - QueryUtils：游戏查询工具
 *    - EventUtils：事件辅助
 *    - RandomUtils：确定性随机数（帧同步友好）
 *    - AnimationUtils：动画时间管理
 *    - MathUtils：数学工具
 */

/**
 * 引擎集成
 * ─────────────────────────────────────────────────────────────────────────
 * 
 * ✓ World 系统
 *   - ClientGameRunner 整合 World
 *   - 利用 World 的 fixedUpdate, update, render 回调
 *   - 渲染精灵位置、旋转、缩放
 * 
 * ✓ Camera2D
 *   - 通过 World.camera 控制视图
 *   - 可扩展实现摄像机跟随
 * 
 * ✓ SpriteManager / Renderer
 *   - 通过 Actor.spriteId 关联精灵
 *   - 在 render 回调中更新精灵状态
 */

/**
 * 配置系统集成
 * ─────────────────────────────────────────────────────────────────────────
 * 
 * ✓ LevelConfig
 *   - SceneManager 根据 LevelConfig 初始化关卡
 *   - 支持阵营、单位、触发器配置
 * 
 * ✓ MapConfig
 *   - 用于获取单位初始位置
 *   - 可扩展支持碰撞、路点等
 * 
 * ✓ UnitConfig / ModelConfig
 *   - Actor 初始化时使用
 *   - 支持技能、属性等配置
 */

/**
 * 工具集成
 * ─────────────────────────────────────────────────────────────────────────
 * 
 * ✓ 可扩展集成现有工具
 *   - BehaviorTree：用于 AI 系统
 *   - AStar：用于寻路
 *   - Quadtree：用于空间划分优化查询
 *   - Timeline：用于动画和事件序列
 */
